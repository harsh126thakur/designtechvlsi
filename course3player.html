Cadence Analog Basics

Course Overview

This course teaches the fundamentals of analog IC design using Cadence Virtuoso. You will learn schematic design, simulation, layout design, and verification using industry-standard tools used in semiconductor companies.

Topics Covered

  • Cadence Virtuoso Environment
  • Schematic Design
  • Analog Circuit Simulation
  • DC / AC / Transient Analysis
  • Layout Design Basics
  • DRC and LVS Verification
