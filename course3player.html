Cadence Analog Basics

Course Description

This course introduces analog circuit design using Cadence Virtuoso. You will learn schematic design, simulation, layout and verification used in semiconductor industry.

Topics Covered

  • Cadence Virtuoso Introduction
  • Schematic Design
  • DC / AC Analysis
  • Transient Simulation
  • Layout Basics
  • DRC and LVS
